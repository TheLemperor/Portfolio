AOI Circuit Design

DE: 11/9/2020 - 12/5/20

The problem of this project was to create a control system that utilized AOI Logic. It was necessary to have each person create a part of the circuit by making a truth table, the equation and then simplifying it down. Once that was done, the multisim simulation could be finished and breadboarding would be started. It was all combined at the end.

For this project, I created the Covid 19 section of the circuit. I spent a large amount of time creating a truth table with 256 different combinations and an equation with 16 minterms. I was able to take out/simplify around 30-40 gates yet it was still huge. Next I moved on to the multisim and took advantage of the fact that I could share the inverters' outputs and utilize XOR gates to make it even simpler. I was then able to create the circuit in parts on the breadboard.

PLD Design

DE: 3/5/2021 - 3/13/2021

The problem of this project was to create an artistic design using the PLD and a logic converter. There had to be documention with a design brief, display definitions, truth table, logic expressions and then the multisim/assembled circuit.

For this project, I created an LED display that could be featured on an electronic game by a company like Hasbro. I first came up with the pattern the LEDs were going to be in, the order they would flash, then the truth table and equations using the logic converter. After that, I was able to create the circuit simulation and assemble the actual circuit. The lights would flash to the right, then to the left, and finally towards the middle.
